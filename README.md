### Setup

- Clone o projeto.

```shell
git@github.com:tsmsp/app-desk.git
```

- Será necessário utilizar [Electron.js](https://www.electronjs.org/).

```shell
npm install electron -g
```

- Execute o código abaixo.

```shell
$ npm start
```
